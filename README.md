全屏循环播放视频和图片的Android应用，适合在安卓广告机里用。
# 全屏媒体播放器

一个基于Kotlin开发的Android应用，支持全屏循环播放视频和图片。通过简洁的交互设计，提供沉浸式的媒体播放体验。因为我不会写代码，所以……它使用了AI。

## 功能特性

- **全屏播放**: 支持视频和图片的全屏循环播放
- **简单操作**: 通过系统或第三方文件管理器选择媒体文件
- **便捷交互**: 三击屏幕即可重新选择文件
- **轻量高效**: 无冗余功能，但是还是有4MB多，我也不知道AI塞了什么屎山代码进去
- **关于系统**：需要安卓5.0以上，应该吧，Android studio好像最低只有5.0了
